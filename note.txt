## Mục 3 - MPI Parallel Version

B1: Biên dịch code:
cd "c:\Users\Men\OneDrive\Máy tính\parallel computing\Adaptive-Huffman-Coding\MPI"
g++ -I"C:\Program Files (x86)\Microsoft SDKs\MPI\Include" -fopenmp mpi_adaptive_huffman.cpp -L"C:\Program Files (x86)\Microsoft SDKs\MPI\Lib\x64" -lmsmpi -o mpi_huffman.exe

B2: Chạy file test:
# Cần thêm MPI vào PATH trước khi chạy:
$env:PATH += ";C:\Program Files\Microsoft MPI\Bin"

# Chạy với số processes khác nhau:
mpiexec -n 2 .\mpi_huffman.exe           # Test với file test.txt, 2 processes
mpiexec -n 2 .\mpi_huffman.exe test      # Test với file test.txt, 2 processes  
mpiexec -n 2 .\mpi_huffman.exe repeat    # Test với file repeat.txt, 2 processes
mpiexec -n 2 .\mpi_huffman.exe 10        # Test với file 10.txt, 2 processes
mpiexec -n 4 .\mpi_huffman.exe 100       # Test với file 100.txt, 4 processes
mpiexec -n 4 .\mpi_huffman.exe 1000      # Test với file 1000.txt, 4 processes
mpiexec -n 8 .\mpi_huffman.exe 10000     # Test với file 10000.txt, 8 processes

## So sánh Performance

Để so sánh hiệu suất, chạy cùng một test với các phiên bản:

2. OpenMP: .\openmp_huffman.exe 100

4. MPI 4 processes: mpiexec -n 4 .\mpi_huffman.exe 100
